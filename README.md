## Sobre el Proyecto

**Minimal Labs** es una colección de ejemplos mínimos funcionales que muestran cómo implementar tareas técnicas esenciales con la menor cantidad de código posible.  

Cada carpeta contiene un ejemplo autocontenido, con dependencias mínimas.
